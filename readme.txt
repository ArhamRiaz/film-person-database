The 'movies.db' is used in this project and is located in the same folder as this readme.txt file.

The custom webpage designed to query this database uses, node.js, express, javascript, html and pug.

To access the webpage open command prompt in this folder and enter 'node server.js' and then open localhost:3000 in your browser.
